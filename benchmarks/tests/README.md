# Benchmark Tests

This directory is reserved for tracked benchmark-owned test fixtures and test contracts.

Canonical benchmark-owned fixture roots live under `benchmarks/tests/fixtures/`:
- `benchmarks/tests/fixtures/bench`
- `benchmarks/tests/fixtures/corpora`
- `benchmarks/tests/fixtures/databases`

The repository-root `tests/fixtures` path is a compatibility symlink that
resolves to this benchmark-owned fixture tree.
