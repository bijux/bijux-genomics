# All-Domain Readiness Proof

This directory stores tracked deterministic readiness proof that spans the governed FASTQ, BAM,
and VCF benchmark surface as one all-domain contract.

`active-stage-catalog.tsv` is regenerated by
`bijux-dna bench readiness render-all-domain-active-stage-catalog` and keeps one governed row per
active FASTQ, BAM, and VCF stage, including active tool coverage, benchmark-ready coverage,
parser coverage, normalized schema presence, and report section coverage. Active scope excludes
`planned`, `future`, and equivalent not-yet-active lifecycle rows.

`active-stage-tool-matrix.tsv` is regenerated by
`bijux-dna bench readiness render-all-domain-active-stage-tool-matrix` and keeps one governed row
per active FASTQ, BAM, and VCF stage-tool binding, including corpus, asset, adapter, parser,
schema, and unified lifecycle status coverage. Active scope excludes `planned`, `future`, and
equivalent not-yet-active lifecycle rows.

`retained-tools.tsv` is regenerated by
`bijux-dna bench readiness render-all-domain-retained-tools` and keeps one retained row per active
tool across the unified stage-tool inventory after planned-only bindings are removed from active
scope.

`no-planned-rows.json` is regenerated by
`bijux-dna bench readiness render-all-domain-no-planned-rows` and proves the governed active
all-domain scope contains no `planned`, `future`, or equivalent not-yet-active lifecycle rows.
