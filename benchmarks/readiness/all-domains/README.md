# All-Domain Readiness Proof

This directory stores tracked deterministic readiness proof that spans the governed FASTQ, BAM,
and VCF benchmark surface as one all-domain contract.

`active-stage-catalog.tsv` is regenerated by
`bijux-dna bench readiness render-all-domain-active-stage-catalog` and keeps one governed row per
active FASTQ, BAM, and VCF stage, including active tool coverage, benchmark-ready coverage,
parser coverage, normalized schema presence, and report section coverage.

`retained-tools.tsv` is regenerated by
`bijux-dna bench readiness render-all-domain-retained-tools` and keeps one retained row per active
tool across the unified stage-tool inventory.
