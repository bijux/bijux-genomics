# Micro Readiness Proof

This directory stores tracked deterministic readiness proof for the governed local micro-benchmark
execution slice.

`MICRO_BENCHMARK_EXECUTION_READY.json` is regenerated by
`bijux-dna bench readiness render-micro-benchmark-execution-ready`. The gate reruns the disposable
`bench run-micro` execution slice under `runs/bench/micro/`, then fail-closes unless Goals
471–479 still pass and the retained FASTQ, BAM, VCF, core germline, aDNA, eDNA, and amplicon
surfaces still keep representative local real execution evidence explicit.
