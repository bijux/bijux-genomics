# VCF Upstream Closure

`science/docs/upstream/vcf/` is the tracked closure-control surface for governed
VCF calling and downstream inference tools in `bijux-genomics`.

[../README.md](../README.md) defines the broader upstream archive boundary, and
[VCF_DOWNSTREAM_CLOSURE_LEDGER.tsv](VCF_DOWNSTREAM_CLOSURE_LEDGER.tsv) keeps
the VCF promotion debt explicit.

## Scope

- supported VCF calling tools whose science authority must remain reviewable
- planned downstream VCF inference families that need citation and acceptance
  contracts before promotion
- placeholder or unresolved upstream identities that must stay visible

## Layout

- [VCF_DOWNSTREAM_CLOSURE_LEDGER.tsv](VCF_DOWNSTREAM_CLOSURE_LEDGER.tsv)
  tracked blocker ledger for governed VCF tools

## Rules

- keep supported calling surfaces separate from planned downstream promotion
- do not promote planned phasing, IBD, imputation, demography, or population
  analysis tools on successful execution alone
- keep placeholder upstream locators explicit until tool identity, wrapper
  contract, and scientific acceptance criteria are reconciled together
