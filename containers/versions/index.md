# Container Versions

Canonical index for container versioning artifacts.

- [../README.md](../README.md) defines the container control surface that owns
  these versioning artifacts.
- [../docs/VERSION_AUTHORITY.md](../docs/VERSION_AUTHORITY.md) defines the
  authority order for editable pins, generated lock material, and source
  provenance.
- [../docs/LOCK_LIFECYCLE.md](../docs/LOCK_LIFECYCLE.md) defines the lock
  generation and drift-check workflow.
- Version pins: [containers/versions/versions.toml](versions.toml)
- Lock/update policy: [containers/versions/LOCK.md](LOCK.md)
- Generated lock artifact: [containers/versions/lock.json](lock.json)
- Version index hash: [containers/versions/index.sha256](index.sha256)
