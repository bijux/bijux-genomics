# CI Stages Configs

Purpose: stage contract catalogs by domain scope.

- `stages.toml`: primary stage catalog
- `stages_vcf.toml`: current VCF baseline stages
- `stages_vcf_downstream.toml`: planned downstream VCF stage backlog

Canonical paths:
- `configs/ci/stages/stages.toml`
- `configs/ci/stages/stages_vcf.toml`
- `configs/ci/stages/stages_vcf_downstream.toml`
