# TOOL_COVERAGE

## Supported outputs
- `samtools` (validate, sort/index, qc)
- `bwa` / `bowtie2` (alignment outputs)
- `picard` / `gatk` (markdup outputs)
- `mosdepth` (coverage summaries)
- `mapDamage2` (damage profiles)
- `pydamage` (contamination/authenticity)
- `rxy` (sex inference)

## Planned / missing
- Additional haplogroup tools (planned, not yet supported).
- Expanded kinship formats (planned, not yet supported).
