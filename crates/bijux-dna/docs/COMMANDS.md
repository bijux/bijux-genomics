# COMMANDS

## Command taxonomy
- `run`: FASTQ pipeline and stage command family.
- `plan`: pipeline planning and explanation command family.
- `bench`: benchmark configuration, suite, publication, and corpus workflows.
- `env`/`registry`: environment and tool-registry inspection.
- `status`: repository and contract status inspection.

## Core commands
- `bijux-dna env`
- `bijux-dna registry`
- `bijux-dna run`
- `bijux-dna plan`
- `bijux-dna bench`
- `bijux-dna status`

See `bijux-dna-api` for contract logic.
