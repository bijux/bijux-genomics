Validate FASTQ reads (pre).

Usage: bijux dna fastq validate-pre [OPTIONS]

Options:
      --list-tools             
      --dry-run                
      --allow-silver           Allow silver-tier tools
      --allow-experimental     Allow experimental-tier tools
      --env <ENV>              
      --sample-id <SAMPLE_ID>  
      --r1 <R1>                
      --out <OUT>              
      --tools <TOOLS>          
      --strict                 
  -h, --help                   Print help

Examples:
  bijux dna fastq validate-pre --r1 reads.fastq.gz --out artifacts --sample-id SAMPLE --tools fastqvalidator
  bijux dna fastq validate-pre --list-tools
