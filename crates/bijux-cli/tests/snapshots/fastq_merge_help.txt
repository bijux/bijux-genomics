Merge paired-end FASTQ reads.

Usage: bijux fastq merge [OPTIONS]

Options:
      --list-tools          
      --dry-run             
      --allow-silver        Allow silver-tier tools
      --allow-experimental  Allow experimental-tier tools
  -h, --help                Print help

Example:
  bijux fastq merge --r1 reads_1.fastq.gz --r2 reads_2.fastq.gz --out artifacts --sample-id SAMPLE --tools vsearch
