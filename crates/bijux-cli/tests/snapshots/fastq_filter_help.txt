Filter FASTQ reads.

Usage: bijux fastq filter [OPTIONS]

Options:
      --list-tools                                           
      --dry-run                                              
      --allow-silver                                         Allow silver-tier tools
      --allow-experimental                                   Allow experimental-tier tools
      --env <ENV>                                            
      --sample-id <SAMPLE_ID>                                
      --r1 <R1>                                              
      --out <OUT>                                            
      --tools <TOOLS>                                        
      --max-n <MAX_N>                                        
      --low-complexity-threshold <LOW_COMPLEXITY_THRESHOLD>  
      --kmer-ref <KMER_REF>                                  
  -h, --help                                                 Print help

Examples:
  bijux fastq filter --r1 reads.fastq.gz --out artifacts --sample-id SAMPLE --tools fastp
  bijux fastq filter --list-tools
