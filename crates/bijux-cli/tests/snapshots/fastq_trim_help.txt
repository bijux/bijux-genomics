Trim FASTQ reads (quality/adapters) and emit canonical outputs.

Usage: bijux fastq trim [OPTIONS]

Options:
      --list-tools
          
      --dry-run
          
      --list-adapter-presets
          
      --list-adapters
          
      --env <ENV>
          
      --sample-id <SAMPLE_ID>
          
      --r1 <R1>
          
      --out <OUT>
          
      --tools <TOOLS>
          
      --adapter-bank-preset <ADAPTER_BANK_PRESET>
          Adapter bank preset name (default: ancientdna-illumina)
      --adapter-bank <ADAPTER_BANK>
          Adapter bank selection: preset:<name> (deprecated; use --adapter-bank-preset)
      --adapter-bank-file <ADAPTER_BANK_FILE>
          Adapter bank file (yaml/json)
      --enable-adapter <ENABLE_ADAPTER>
          
      --disable-adapter <DISABLE_ADAPTER>
          
  -h, --help
          Print help

Example:
  bijux fastq trim --r1 reads.fastq.gz --out artifacts --sample-id SAMPLE --tools fastp
